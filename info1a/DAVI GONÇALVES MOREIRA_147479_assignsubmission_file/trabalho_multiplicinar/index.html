Trabalho multidisciplinar

Sobre o trabalho

É um trabalho interdisciplinar (abrange mais de uma disciplina), sobre educação financeira, o trabalho consiste em criar um site (Desenvolvimento FRONT END 1) que explique sobre o assunto, o trabalho tem os seguintes elementos de outras matérias: Cards sobre educação financeira (Ferramentas de Desenho) e uma planilha de custos sobre energia elétrica (Matemática 1). Os dois elementos serão melhores desenvolvidos em suas respectivas páginas, confira nos links abaixo.
