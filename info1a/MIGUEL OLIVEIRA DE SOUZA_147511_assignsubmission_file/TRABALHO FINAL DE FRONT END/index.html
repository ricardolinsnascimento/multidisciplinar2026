Trabalho Multidisciplinar - Educação Financeira

Sobre o Trabalho

Este trabalho foi produzido por estudantes do IFMS Campus Dourados como parte de uma atividade multidisciplinar sobre Educação Financeira. O projeto integrou conhecimentos de diferentes disciplinas, permitindo o desenvolvimento de habilidades relacionadas ao planejamento financeiro, análise de gastos, produção textual e criação de materiais visuais. O objetivo é conscientizar sobre a importância da educação financeira para uma melhor organização dos recursos e para a tomada de decisões mais responsáveis no dia a dia.

Disciplinas Envolvidas

Matemática: elaboração de uma planilha de custos para controle financeiro.

Língua Portuguesa: produção de texto explicativo sobre os dados da planilha.

Ferramentas de Desenho: criação de uma imagem relacionada ao tema educação financeira.
