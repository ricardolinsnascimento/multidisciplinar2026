Trabalho Multidisciplinar - Educação Financeira

Qual o objetivo desse trabalho?

Este trabalho multidisciplinar uniu conhecimentos de Front-End, Ferramentas de Desenho e Matemática. O projeto envolveu a criação de um site, o desenvolvimento de elementos visuais e a análise do consumo de energia elétrica, promovendo a educação financeira dos alunos.
