Trabalho Multidisciplinar - Educação Financeira

Sobre o Trabalho

Este trabalho multidisciplinar teve como objetivo desenvolver conhecimentos sobre educação financeira utilizando conteúdos de diferentes disciplinas.

Na disciplina de Matemática da Professora Cristiane Bender, foi elaborada uma planilha de custos para analisar gastos.

Na disciplina de Ferramentas de Desenho da Professora Fabricia, foi criada uma imagem relacionada a educação financeira para representar visualmente o tema estudado.
