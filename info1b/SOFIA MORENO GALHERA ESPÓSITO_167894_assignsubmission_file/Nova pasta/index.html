Trabalho Multidiciplinar Educação Financeira

A educação financeira é uma forma de saber essencial para o dia a dia na sociedade atual, afetando 80,4% da população brasileira, mesmo assim é uma area muito pouco estudada e de dificil acesso, pela necessidade desta educação é otimo que o IF esteja trabalhando para entregar esta materia para os alunos.

Mas por quê este é um assunto tão importante?

A educação financeira é importante pois ela é quem te ajuda a adultos fazerem escolhas economicas conscientes e que vão beneficiar a querida carteira de todos, é importante ensinar essa abilidade à jovens o mais cedo possivel para que eles possam cometer erros no momento em que erros podem ser cometidos.
