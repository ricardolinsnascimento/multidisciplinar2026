Trabalho Multidisciplinar - Educação Financeira

Sobre o trabalho

Este trabalho multidisciplinar teve como objetivo aprender sobre educação financeira utilizando conhecimentos de diferentes disciplinas.

Na disciplina de Matemática foi desenvolvida uma planilha de custos para organização financeira.

Em Ferramentas de Desenho foi criada uma imagem relacionada ao tema.
