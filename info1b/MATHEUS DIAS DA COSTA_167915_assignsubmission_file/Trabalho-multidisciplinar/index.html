Trabalho Multidisciplinar - Educação Financeira

Sobre o Trabalho

Este trabalho multidisciplinar teve como objetivo ensinar conceitos de Educação Financeira utilizando conhecimentos de várias disciplinas do curso. Cada matéria contribuiu de forma importante para o desenvolvimento da atividade.

  • Matemática: elaboração da planilha que calcula o gasto de energia.
  • Ferramentas de Desenho: criação de cards para instagram.
  • Front-End: desenvolvimento deste site utilizando HTML e CSS.
